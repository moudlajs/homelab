| Model | RAM Usage | Speed | Gaming Impact |
|-------|-----------|-------|---------------|
| Llama 3.2 3B | ~3-4 GB | Fast | Minimal |
| Mistral 7B | ~6-8 GB | Medium | Noticeable |
| Llama 13B | ~12 GB | Slow | Can't game |

When NOT querying: 0 CPU, RAM stays allocated.
Run `ollama stop` to free RAM for gaming.

## Tailscale vs WireGuard

| Aspect | Tailscale | WireGuard |
|--------|-----------|-----------|
| Port forwarding | Not needed | Required |
| Router access | Not needed | Required |
| Setup | 2 minutes | 30+ minutes |
| Security | Excellent | Excellent |
| Speed | Fast | Slightly faster |

Tailscale is recommended for homelab use.
